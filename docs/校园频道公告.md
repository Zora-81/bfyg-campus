# 📢 校园频道上线啦！给一高同学的一份使用指南

哈喽各位一高的同学，欢迎来到「校园频道」！这地方说白了就是咱学校自己的小天地——不用翻墙、不用加一堆群，一个账号就能进来唠嗑、找人、看通知。想吐槽食堂阿姨的手抖，想组队刷题，想找同好聊动漫游戏，都成。

进来先别懵，给你捋捋都能干点啥：

**聊天吹水**：左边抽屉里藏着一堆频道，综合大厅随便侃，各个兴趣频道各聊各的。消息是实时推送的，你发完对方秒收，不用来回刷新。

**找人私聊**：右上角铃铛切到「好友」，能搜 uid 或昵称加好友、收好友请求。加完直接从好友列表点进去开小窗私聊，悄悄话就这儿说。

**@提醒**：群里想叫谁，打 @ 加他名字就高亮提醒，保证他装不了没看见。

**点赞评论转发**：每条消息底下都能点赞、评论、引用回复，还能转发。觉得哪条有意思，爱心怼上去，楼主能看到小特效。

**热门话题**：右边栏有个实时热度榜，按评论数加时间衰减算的，谁家瓜最热一目了然，点一下直接跳过去吃瓜。

**频道推荐**：不知道进哪个？右侧给你推了几个频道，挑顺眼的进。

**改头换面**：点左上角头像进设置，背景能换星空、白昼、森绿，还能传自己的壁纸调模糊和暗角，整成你喜欢的样子。

**学校介绍**：侧栏点校徽，校徽动画 + 咱学校 1956 年建校那一堆介绍，新同学可以补补课。

对了，公告频道只有管理员能发，普通同学安心当吃瓜群众就行。有啥想说的、想加的频道，找管理员唠嗑。

青春不散场，我们在校园频道等你～
